Creative technologist. I write code, music, and voice characters — usually for the same project.

Seventeen years self-taught. The work is generative art systems, audio tooling, and agentic infrastructure built to run my own creative practice — not someone else's product.

WaveLoop is my music software line: tape loopers for Reaper (x1, x4, Choppa) that prioritize feel over precision. No quantize, no timeline, no perfection.

Under Lucian Labs, I build in two directions. Outward-facing creative tools: banger (browser noise synthesis), sketchy (generative drawing frameworks), lewps, super-mouse, numbaz. Inward-facing infrastructure: enzyme (LLM-optimized codebase digests for Claude Code), signal-chain (AI audio orchestration for known hardware), scope-drift (session context monitor for Claude Code), GroundControl (cross-machine agent coordination). On the personal side: ThinAmp is a desktop audio player I built for myself, and relay routes OSC and WebSocket between Ableton and everything else I run live.

The lab is at lucianlabs.ca — ongoing experiments in agentic tooling, generative systems, and creative infrastructure.
